내 프로젝트

항공권 예약 웹사이트

Kotlin, HTML, CSS, Javascript, JSP 사용자에게 항공기 및 운항 정보를 제공하고, 편리한 항공기 예약 서비스를 제공하는 항공예매 시스템을 제작했습니다. 이를 위해 한국 공항 공사에서 제공하는 open API 의 항공기 운항 정보를 이용합니다. 프로젝트 진행을 위해 주기적인 스프린트 회의를 진행하며, 요구사항분석, 상세 설계, 구현의 단계를 거쳐 웹 사이트 결과물을 도출합니다. 예약 기능을 중심으로 로그인, 운항정보, 예약 안내 등의 기능을 제공합니다. UML 다이어그램과 디자인 패턴을 활용하여 프로젝트를 진행했습니다.
데모 영상 깃허브

3D 객체 인식 딥러닝 프로젝트

Python, Point Cloud, CloudCompare Point Cloud 3D 객체를 딥러닝하여 분류하는 딥러닝 프로젝트에 참여했습니다. SamsungLabs의 FCAF3D에 DTaaS의 데이터셋을 학습켰습니다. ISTQB 자격증을 취득하며 얻은 지식을 토대로 전처리 코드를 정적분석하여 에러를 해결했습니다. 데이터 부족으로 인한 overfitting을 해결하기 위해 CloudCompare를 통해 데이터에 라벨링했습니다. 이후 연계 해커톤에서 3D 데이터를 활용한 경찰 수사 방안을 발표하여 수상했습니다.
FCAF3D 깃허브

신한DS 인턴

외환/자금 업무와 SWIFT 신전문 전환 프로젝트를 담당했습니다. Proframe과 pro-C를 사용하여 서비스를 개발하고 WebSquare와 JavaScript 및 HTML5를 활용하여 단말화면을 개발했습니다. 서비스 및 단말화면을 구현하여 로컬, 개발, 테스트 단계 후에 운영 서버에 반영했습니다. 데이터베이스 변경시 현지 직원의 의도를 우선하되 META 시스템 규칙을 준수했습니다. 외환 송금, 수출입, FX 거래, 머니마켓 거래의 프로세스를 단말화면, 데이터, 회계 변화를 중심으로 도식화하여 발표했습니다.
